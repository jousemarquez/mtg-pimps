# 🎴 MTG Pimp Collection Manager

Professional dashboard for managing and scoring premium Magic: The Gathering collections.

## 🚀 Experience the "Pimp" Status

This application allows collectors to register their cards and automatically calculate their "Pimp Score" based on premium attributes like Japanese prints, Foil/Holo versions, Artist Signatures, and Alters.

### 🔥 Features
- **Modern Dashboard**: Track your collection completion, ownership, and Pimp Ratio at a glance.
- **The Vault**: Advanced collection table with inline editing, dynamic search, and "Pimp" filtering.
- **Scryfall Integration**: Autocomplete card and set names using the Scryfall API.
- **Advanced Scoring**: 
  - `PIMP MAX`: The holy grail (Owned + Japanese + Signed).
  - `PIMP`: Premium aesthetic level based on accumulated attributes.
- **Glassmorphism Design**: High-end visual style inspired by modern TCG tools and Notion.

## 🛠 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4
- **State Management**: Zustand (with LocalStorage persistence)
- **Icons**: Lucide React
- **Animations**: Framer Motion

## 📦 Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open the app**:
   Navigate to [http://localhost:3000](http://localhost:3000)

## 🚢 Deployment

Compatible with **Vercel** out of the box. Simply connect your GitHub repository and deploy.
